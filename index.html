Mes projets

Projet pour rendre dynamique le site web d'une photographe :

Dance ce projet développé en HTML, CSS et JavaScript, j'ai créé une page interactive. Divisé en étapes clés, il comprend l'installation de l'environnement, la création d'une galerie de travaux dynamique avec filtrage par catégorie, l'ajout d'une fenêtre modale pour l'administration des projets. L'objectif étant de créer une application web fonctionnelle.

Projet pour créez une application web de location immobilière :

Dans ce projet que j'ai initialisé avec Create React App et l'intégration de React Router, je me suis concentré sur la création de composants et de pages spécifiques, telles que la page d'accueil et la page de logement avec un carrousel. Le projet met l'accent sur des fonctionnalités comme la gestion des erreurs d'ID de propriété. J'y ai utilisé Sass pour la mise en page et pu réutiliser judicieusement les composants. La vérification finale garantit la conformité aux exigences, le respect du design de la maquette, et la gestion efficace des erreurs.

Projet pour implémentez le front-end d'une application bancaire :

Dans ce projet j'ai dû commencer par l'installation du back-end en local et vérifier les routes de connexion. Ensuite, j'ai créé l'application React avec React Router, puis j'ai introduis Redux pour gérer l'état global, y compris la connexion de l'utilisateur. Ensuite il a fallut mettre à jour le nom d'utilisateur via un formulaire sécurisé. Enfin, j'ai créé les endpoints nécessaires dans le fichier Swagger, pour ajouter modifier ou visualiser les transactions.
